# GuardianAI

GuardianAI is an AI-powered desktop security assistant that helps users understand whether unusual behavior on their Windows computer is normal or potentially suspicious.

Instead of only showing technical statistics, GuardianAI explains what is happening in simple language and warns users about possible security risks.

## Planned Features

- Monitor CPU, RAM, Disk and Battery
- Detect unusual system behavior
- Monitor network activity
- Detect suspicious processes
- Explain possible reasons for issues
- Learn normal user behavior
- Provide AI-based recommendations
